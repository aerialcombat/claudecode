HTMX Components Demo

한글 최적화 HTMX 컴포넌트 데모 페이지

1. Live Search (실시간 검색)

입력하면서 실시간으로 검색 결과를 표시합니다. 한글 IME 처리가 포함되어 있습니다.

💡 한글 IME 처리
이 컴포넌트는 한글 조합 중에는 검색을 트리거하지 않고, 조합이 완료된 후에만 검색합니다. compositionstart와 compositionend 이벤트를 활용합니다.
검색 결과가 여기에 표시됩니다

2. Infinite Scroll (무한 스크롤)

스크롤하면 자동으로 더 많은 콘텐츠를 불러옵니다. hx-trigger="revealed"를 사용합니다.

콘텐츠 항목 1

스크롤 내리면 더 많은 항목이 로드됩니다

콘텐츠 항목 2

각 항목은 독립적으로 렌더링됩니다

콘텐츠 항목 3

무한 스크롤은 사용자 경험을 향상시킵니다

⬇️ 스크롤하여 더 불러오기

3. Modal (모달)

동적으로 콘텐츠를 로드하는 모달 컴포넌트입니다. 서버에서 HTML을 받아 표시합니다.

4. Delete Confirmation (삭제 확인)

삭제 전에 확인 다이얼로그를 표시합니다. hx-confirm 속성을 사용합니다.

삭제 가능한 항목

이 항목을 삭제할 수 있습니다

5. Inline Edit (인라인 수정)

클릭하여 바로 수정할 수 있는 필드입니다. 수정 모드와 표시 모드를 전환합니다.

수정 가능한 텍스트

📚 사용 가능한 컴포넌트

🟢 Simple Components (간단한 컴포넌트)

  • inline-edit.html - 클릭하여 수정
  • delete-confirm.html - 삭제 확인
  • toggle.html - 표시/숨김 전환
  • boosted-links.html - 프로그레시브 향상
  • live-badge.html - 자동 업데이트 뱃지

🟡 Medium Components (중급 컴포넌트)

  • live-search.html - 실시간 검색
  • pagination.html - 페이지네이션
  • tabs.html - 탭 네비게이션
  • dropdown.html - 드롭다운 메뉴
  • sort-filter.html - 정렬/필터

🔴 Complex Components (고급 컴포넌트)

  • infinite-scroll.html - 무한 스크롤
  • modal.html - 모달 다이얼로그
  • form.html - 다단계 폼
  • autocomplete.html - 자동완성
  • drag-drop.html - 드래그앤드롭 정렬

🔧 Go Handlers

  • search_handler.go.tmpl - 검색 핸들러
  • content_handler.go.tmpl - CRUD 핸들러
  • htmx_helpers.go.tmpl - 유틸리티 함수

🚀 사용 방법

1. 컴포넌트 생성

python scripts/generate_component.py \
  --type search \
  --name topic-search \
  --korean \
  --go-handler \
  --output /path/to/project/

2. 템플릿 분석

python scripts/analyze_templates.py \
  --path /path/to/templates/ \
  --priority

3. 테스트 페이지 생성

python scripts/test_component.py \
  --component search \
  --output test-search.html
